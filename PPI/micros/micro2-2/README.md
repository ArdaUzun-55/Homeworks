# Mikroübung 2-2
Das Programm aus Mikroübung 2-1 soll nun so erweitert werden, dass drei
Ganzzahlen von der Standardeingabe gelesen werden.  Anschließend soll die Summe
aus den drei eingelesenen Zahlen berechnet und ausgegeben werden.

Beispiel:

|Input  |Output |
|-------|-------|
|1 2 3  |6      |
|-3 6 2 |5      |


__Hinweis:__
Nach jeder Änderung müssen Sie Ihr Programm neu mit dem Befehl `make` kompilieren. 
Mit 
```bash
./solution < sample.in | diff - sample.out
```
können Sie Ihre Änderungen lokal auf Ihrem PC überprüfen. Sobald die Änderungen den gewünschten Effekt erzielen, erstellen Sie einen `Commit` und `pushen` diesen.
